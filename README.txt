Запуск
	Используем git и docker(убедитесь, что докер запущен)
	выполняем в консоль: git clone https://github.com/apple5343/golangProject
	переходим в папку с проектом и выполняем(обратите внимание на точку в конце): docker build -t myapp .
	затем выполняем: docker run -p 8080:8080 -d -v <путь до проекта>/database:/app/database -v <путь до проекта>/config:/app/config myapp
		Если путь будет указан неверно, то изменения не будут сохраняться в базу данных при перезапуске
	сервер будет запущен по адресу http://localhost:8080/(Если вы изменяете порт в конфиге, то указывайте его в правой части после ":" в -p 8080:8080, в левой части можно изменить порт докера)
	после можно прописать docker ps для достоверности


	для остановки набираем docker ps, берем CONTAINER ID и пишем docker stop <CONTAINER ID>


Функции приложения
	Допустимые символы: -+/*, можно использовать десятичные числа типа 1.5, деление на 0 выдает ошибку
	При перезапуске будут выполнятся прерванные операции с последнего состояния
	Можно удалять воркеры
	Можно просмотреть подробную информацию о задаче
